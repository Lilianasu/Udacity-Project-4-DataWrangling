The next project is the result of the Udacity Data Analyst Udacity Nanodegree  Wrangle and Analyze Project. It consists on gathering tweet data from the WeRateDogs profile, performing  quality and tidiness assessments, and cleaning it according to the necessities of the visualizations.

It comprehends the following parts:

1. wrangle_report.html
A brief explanation of the assessment and the wrangling processes performed.

2. wrangle_act.ipynb
Jupyter notebook in which the data is gathered in three ways: directly, through a web server, and through Twitter's API Tweepy. Also it is performed the wrangling process.

3. act_report.html
The analysis and visualization resulting from the wrangling.
